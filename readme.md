# Autocompletion :

Projet d'autocomplétion (thème: animaux). Utilise PHP + MySQL + Tailwind (Play CDN) pour l'UI.

## Installation :
1. Importer `database.sql` dans MySQL.
2. Mettre `config.php` à jour avec tes identifiants.
3. Déployer les fichiers sur Plesk à la racine du domaine.
4. Partager le repo avec `deepthoughtlaplateforme` et ajouter un lien sur les pages.
